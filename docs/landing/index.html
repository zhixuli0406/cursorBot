選擇 Docker 獲得最簡單的安裝體驗，或本地安裝以獲得更多控制權。

3

啟動服務

4

連接您的 Bot

加入您的 Telegram/Discord Bot Token 即可開始對話！

終端機 
# 複製程式碼庫
git clone https://github.com/zhixuli0406/cursorBot.git
cd cursorBot

# 設定環境變數
cp env.example .env
nano .env  # 填入您的 API 金鑰

# 方式一：Docker（推薦）
docker compose up -d --build

# 方式二：本地 Python
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python -m src.main

功能特色

強大功能

您需要的一切，提升開發工作流程效率。

語音喚醒 & 自然對話

說「Hey Cursor」或自訂喚醒詞啟動助手。支援離線喚醒（Vosk/Porcupine）、 聲紋識別、多語言對話和情緒語調回應。

智慧意圖識別

理解代詞指代（「它」「這個」）、多輪對話上下文、 槽位填充補全缺失資訊、支援中英日混合語音指令。

智慧家居控制

整合 HomeKit、Google Home、小米 IoT。 語音控制燈光、空調、插座等智慧設備。

會議助手

會議錄音、即時轉錄、自動摘要生成、 行動項目提取。讓會議更有效率。

無障礙支援

螢幕閱讀器整合（VoiceOver/NVDA）、純語音導航、 音效/觸覺回饋、為視障用戶優化。

離線能力

離線語音辨識（Vosk）、離線 TTS（Piper/eSpeak）、 本地意圖識別、網路斷線自動切換離線模式。

API 文件

RESTful API & WebSocket

完整的 API 存取，輕鬆整合您的應用程式。

REST 端點

GET /health 健康檢查
GET /health/detailed 詳細狀態
GET /api/nodes 已連接裝置
GET /api/usage 使用統計
POST /api/broadcast 廣播訊息
WS /ws/node 原生應用 Gateway
WebSocket 範例 
// 連接 Gateway
const ws = new WebSocket(
  'ws://localhost:8000/ws/node'
);

// 發送聊天訊息
ws.send(JSON.stringify({
  type: 'chat',
  payload: {
    message: '你好 AI！'
  }
}));

// 建立畫布
ws.send(JSON.stringify({
  type: 'canvas',
  payload: {
    action: 'create'
  }
}));
